Цей проект є системою моніторингу сонячних електростанцій, яка дозволяє відстежувати дані з різних сенсорів,
таких як фотосенсори, температурні сенсори, сенсори напруги та сенсори струму. Система забезпечує збереження
даних у базі даних PostgreSQL, регулярне оновлення даних та відображення їх у реальному часі на веб-дешборді.

Технології

	•	FastAPI: Використовується для створення веб-серверу і реалізації WebSocket для передачі даних у реальному часі.
	•	SQLAlchemy: Використовується для ORM (Object-Relational Mapping) та взаємодії з базою даних PostgreSQL.
	•	Pydantic: Використовується для валідації даних та створення схем для API.
	•	JavaScript: Використовується для обробки даних, що надходять через WebSocket, і динамічного оновлення HTML-дешборду.
	•	HTML/CSS: Використовуються для створення інтерфейсу користувача.

Архітектура

	1.	main.py:
	•	Створює FastAPI додаток.
	•	Налаштовує CORS (Cross-Origin Resource Sharing).
	•	Ініціалізує дані при запуску додатка.
	•	Регулярно генерує тестові дані кожну хвилину.
	•	Відправляє оновлені дані всім підключеним клієнтам через WebSocket.
	2.	database.py:
	•	Налаштовує підключення до бази даних PostgreSQL.
	•	Створює engine та сесію для взаємодії з базою даних.
	•	Визначає базовий клас для моделей SQLAlchemy.
	3.	models.py:
	•	Визначає моделі бази даних для сонячних електростанцій та сенсорів (фото, температура, напруга, струм).
	4.	schemas.py:
	•	Визначає схеми для Pydantic, які використовуються для валідації даних та створення API.
	5.	data_generator.py:
	•	Генерує випадкові показники для сенсорів.
	•	Ініціалізує тестові сонячні електростанції в базі даних.
	6.	index.html:
	•	HTML-дешборд для відображення даних сенсорів.
	•	Підключається до WebSocket сервера для отримання даних у реальному часі та оновлення таблиць.
	7.	styles.css:
	•	Слугує покращенням дизайну проєкту.
